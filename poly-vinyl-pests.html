Poly-Vinyl Pests!

Team Size: 4 members
Duration: 3 months
Engine: Unity

Project Overview

Poly-Vinyl Pests! is an innovative pest control game where players must protect precious vinyl records from music-loving insects. As the Lead Programmer, I was responsible for implementing core gameplay mechanics, enemy AI behavior, and sound integration systems.

Technical Achievements

Developed a dynamic sound reaction system where insects respond to music being played, created an efficient pest spawning system that scales with difficulty, and implemented a physics-based movement system for realistic insect behavior.

Challenges & Solutions

One of the main challenges was optimizing the audio processing system to handle multiple sound sources without impacting performance. This was solved by implementing a custom audio pooling system and utilizing Unity's Job System for parallel processing of audio data.
